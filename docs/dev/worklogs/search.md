## 2026-07-03 21:00 · module-dev · M-01
**Сделано:** `search/rag_demo.py` (297 строк) — сквозной путь «вопрос → ответ с цитатами» без роутера: `embed_texts()` (переиспользован, не продублирован) → `vector_search_chunks()` (bolt, `CALL db.index.vector.queryNodes('chunk_embedding_idx', …)` + `(:Document)-[:HAS_CHUNK]->(:Chunk)`, только чтение) → `call_answer_llm()` (Ollama `/v1/chat/completions`, ANSWER_MODEL) → `answer_question()` собирает `contracts.Answer`, citations строит код из найденных чанков (doc_id/chunk_id/title/year/quote≤300). CLI: `python -m ariadna.search.rag_demo "вопрос" [--top-k] [--json]`.
**Решения:** живой стенд показал, что Ollama отдаёт thinking в отдельном поле `message.reasoning`, а не в `<think>` внутри `content` — но код всё равно страхуется regex-вырезкой `<think>…</think>` из content на случай другой версии/конфигурации; ANSWER_MAX_TOKENS=8192 (замер: тривиальный вопрос уже потратил ~1500 ток. на thinking); свой no-proxy opener (тот же факт про HTTP_PROXY→502, что и в embeddings.py); нет чанков → found=False «в корпусе не найдено» (инвариант №6); LLM-синтез упал, но чанки есть → found=True с честным текстом ошибки + реальные citations (цитаты не теряются). Новые коды SEARCH-004 (векторный поиск), SEARCH-005 (answer-LLM/пустой content) в ERRORS.md.
**Проблемы:** нет — импорт, lint, живой смоук прошли с первого прогона.
**Живой смоук:** эталонный вопрос жюри №2 (циркуляция католита при электроэкстракции никеля) → осмысленный русский ответ со ссылками [1]…[8] на фрагменты + честная оговорка «универсальной оптимальной скорости в источниках нет»; 8 реальных citations (doc_id/chunk_id/title/year/quote) из ИТС НДТ 12-2016 и Обзора техрешений 2024/2025; лог `logs/pipeline/rag_20260703T205513.jsonl` — query_received → answer_ready.
**Открыто:** это будущий `rag_fallback` для роутера A-10 (интерфейс `answer_question(question, top_k)` уже пригоден для вызова из router/retrieval); синтез с reasoning-моделью занимает ~2–3 мин на вопрос — для A-11 стоит обсудить бюджет/таймаут перед демо жюри.

# Паспорт модуля: ui

**Назначение:** Streamlit-интерфейс демо: чат «вопрос → ответ с цитатами»,
визуализация подграфа ответа, карта пробелов ⭐, фильтры гео/год,
блок «Рекомендации» (У-1), литобзор с таблицей сравнения (У-2).

**Вход:** Answer, GapReport, LitReview, list[Recommendation] — вызовами функций
search/analytics напрямую (api-модуль — Could, не ждать его).
**Выход:** экраны Streamlit (код в `ui/`, вне пакета src/ariadna).

**Контракты:** только читает модели из `contracts.py`; своих структур не заводит.

**Зависимости:** streamlit; для графа — streamlit-agraph или pyvis (выбор за
module-dev: критерий — подсветка рёбер цветом и клик по узлу); search, analytics.

**Не входит в зону ответственности:** любая бизнес-логика (запросы, синтез,
аналитика) — UI только отображает готовые модели; запись в Neo4j.

**Известные ограничения:**
- Связи contradicts в подграфе — красным + строка «⚠ противоречие» в ответе (У-3).
- found=False отображать честно: «в корпусе не найдено» + ссылка на карту пробелов.
- Пользователь — исследователь без подготовки в графовых БД: никаких Cypher/термина
  «чанк» в интерфейсе; цитаты — «документ, страница/фрагмент».
- Демо смотрят жюри: приоритет — читаемость и отзывчивость на 4 эталонных запросах,
  не универсальность.
